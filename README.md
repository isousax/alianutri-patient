# AliaPatient — App do Paciente

App mobile (iOS + Android) para pacientes acompanharem seus planos alimentares, orientações e consultas.

## Stack

- **Expo SDK 52** (managed workflow)
- **React Native 0.76** + TypeScript
- **Expo Router** (file-based navigation)
- **React Query 5** (data fetching + cache)
- **NativeWind 4** (Tailwind CSS for RN)
- **Zustand** (state management)
- **Lucide React Native** (icons)
- **Expo Secure Store** (token storage)

## Setup

```bash
npm install
npx expo start
```

Scan the QR code with Expo Go (iOS/Android) or press `a` for Android / `i` for iOS simulator.

## Env vars

Create `.env` with:

```
EXPO_PUBLIC_API_URL=https://api.alianutri.com.br
```

## Structure

```
app/                    # Expo Router screens
  _layout.tsx           # Root layout (fonts, providers)
  index.tsx             # Auth redirect
  login.tsx             # Access code login
  (tabs)/               # Bottom tab navigator
    _layout.tsx         # Tab bar config
    index.tsx           # Home dashboard
    meal-plan.tsx       # Active meal plan viewer
    guidelines.tsx      # Guidelines list
    profile.tsx         # Patient profile + settings
src/
  hooks/                # React Query hooks
  services/             # API client
  stores/               # Zustand stores
  theme/                # Colors, design tokens
  types/                # TypeScript types
assets/                 # Icons, splash, images
```

## Auth Flow

No OAuth. Patients access via a link `/p/<access_code>` generated by the nutritionist in alianutri-web.
The access code is exchanged for a JWT via `POST /patient-portal/auth`.
